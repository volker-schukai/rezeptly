
# Sour Cream

## Vorbereitung

|   Menge | Zutat           |
| -------:| --------------- |
|   500 g | Magerquark      |
|   200 g | Crème fraîche   |
|       2 | Knoblauch Zehen |
|       1 | Zwiebel         |
|    3 EL | Essig           |
|    1 TL | Zucker          |
|    1 TL | Salz            |
| 1 Prise | Pfeffer         |
| 1 Prise | Oregano         |
|    25 g | Schnittlauch                |

## Zubereitung

3 Knoblauch Zehen, 1 Zwiebel und 25 g Schnittlauch klein schneiden

Etwas Oregano dazugeben.

500 g Quark, 200 g Crème fraîche, 3 EL Essig, 1 TL Salz, 1 TL Zucker vermischen.

Alles vermischen und kaltstellen.
